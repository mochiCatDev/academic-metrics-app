  1. Promedio y Media Aritmética

    Explicación

    La media y el promedio son exactamente lo mismo.
    En el día a día usamos la palabra "promedio", pero en el mundo de las matemáticas y la estadística, los profesionales prefieren llamarle "media" (o formalmente, media aritmética). Es el valor que se obtiene al sumar todos los datos de un conjunto y luego dividir el resultado entre el número total de datos.

    Fórmula

    $$\bar{x} = \frac{\sum_{i=1}^n x_i}{n}$$

    ¿Qué significa cada símbolo?

    • \(\bar{x}\) (se lee "x barra"): Representa la media aritmética o promedio.
    • \(\sum\) (letra griega Sigma): Es el símbolo de sumatoria, lo que significa que debes sumar todos los valores desde el primero (\(i=1\)) hasta el último (\(n\)).
    • \(x_i\): Representa cada uno de los valores individuales en tu conjunto de datos.
    • \(n\): Es el número total de datos que estás sumando.

    ¿Cómo lo haces tú mismo?

    Sigue estos pasos para que no te pierdas al momento de quererlo hacer tú mismo en un cuaderno:

    Imaginemos que los datos que tienes son: \([8, 9, 7]\)

    1. Suma todos los datos -> \(8 + 9 + 7 = 24\)
    2. Cuenta la cantidad de tus datos -> \([8, 9, 7] = 3\) datos
    3. Ahora divides la suma total entre la cantidad de datos -> \(\frac{24}{3} = 8\)
    4. ¡Listo! Ya tienes tu media aritmética (promedio), que es por supuesto \(8\)

    Ejemplos

    Ejemplo 1: El promedio de tus calificaciones

    Imagina que quieres saber cuál fue tu rendimiento final en una materia y tus notas de los cuatro periodos del año fueron: 8, 9, 7 y 10.

    1. Sumas todas las notas: \(8 + 9 + 7 + 10 = 34\)
    2. Divides entre el número total de notas: \(\frac{34}{4} = 8.5\)
    3. Resultado: La media aritmética de tus calificaciones es \(8.5\)
    Ejemplo 2: Los puntos anotados por un jugador de baloncesto

    Un jugador de básquetbol quiere saber cuántos puntos mete por partido en promedio. En sus últimos 5 juegos anotó: 12, 18, 15, 20 y 10 puntos.

    1. Sumas los puntos de todos los partidos: \(12 + 18 + 15 + 20 + 10 = 75\)
    2. Divides entre el número total de partidos: \(\frac{75}{5} = 15\)
    3. Resultado: La media aritmética es \(15\) puntos por partido.
    Ejemplo 3: El dinero que gastas en la semana

    Quieres calcular cuánto gastas típicamente en transporte de lunes a viernes. Tus gastos diarios fueron: $2.50, $3.00, $2.50, $4.00 y $3.00.

    1. Sumas el dinero de los 5 días: \(2.50 + 3.00 + 2.50 + 4.00 + 3.00 = 15.00\)
    2. Divides entre los 5 días de la semana: \(\frac{15.00}{5} = 3.00\)
    3. Resultado: La media aritmética de tu gasto diario es de \(\$3.00\)

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function calcularMedia(arr) {
  // 1. Sumamos todo
  let suma = 0;
  for (let i = 0; i < arr.length; i++) {
    suma += arr[i];
  }
  // 2. Dividimos por la cantidad de elementos
  const media = suma / arr.length;
  return media;
}

  2. La Mediana

    Explicación

    La mediana es el valor central de un conjunto de datos ordenados de menor a mayor. Divide la distribución exactamente por la mitad, dejando un 50% de los valores por debajo y un 50% por encima. A diferencia de la media, no se ve afectada por números extremadamente grandes o pequeños, lo que la hace una medida de tendencia central muy confiable en estadística.

    Fórmula Matemática

    La posición de la mediana depende de si el número total de datos (\(n\)) es par o impar:

    • Si \(n\) es impar, la mediana es el dato en la posición: $$\text{Posición} = \frac{n + 1}{2}$$
    • Si \(n\) es par, la mediana es el promedio de los datos en las posiciones: $$\frac{x_{\left(\frac{n}{2}\right)} + x_{\left(\frac{n}{2} + 1\right)}}{2}$$

    Cómo Hacerlo Tú Mismo

    Sigue estos pasos paso a paso en tu cuaderno para calcularla sin errores:

    • Paso 1: Ordena todos tus datos de menor a mayor de forma obligatoria.
    • Paso 2: Cuenta la cantidad total de datos (\(n\)) que tienes en el conjunto.
    • Paso 3: Si la cantidad es impar, la mediana es exactamente el número que se encuentra en el centro de la lista.
    • Paso 4: Si la cantidad es par, toma los dos números del centro, súmalos y divídelos entre \(2\).

    Ejemplos

    • Ejemplo con cantidad impar: Supongamos que tienes las edades \(\{3, 5, 8, 10, 11\}\). Como están ordenadas y son 5 datos (impar), el número del centro es directamente el \(8\).
    • Ejemplo con cantidad par: Supongamos que tienes las notas \(\{2, 4, 6, 8\}\). Al ser 4 datos (par), los dos valores centrales son \(4\) y \(6\). Su promedio es: $$\frac{4 + 6}{2} = 5$$

    Cómo Programarlo

    Así es como una computadora procesa este algoritmo usando JavaScript :

    function calcularMediana(listaNumeros) {
  // 1. Ordenar los datos de menor a mayor sin alterar la lista original
  const datosOrdenados = [...listaNumeros].sort((a, b) => a - b);
  const n = datosOrdenados.length;

  // 2. Encontrar el índice de la mitad exacta
  const indiceMitad = Math.floor(n / 2);

  // 3. Si la cantidad de datos es IMPAR, el del centro es la mediana
  if (n % 2 !== 0) {
    return datosOrdenados[indiceMitad];
  }
  // 4. Si es PAR, se promedian los dos elementos del centro
  else {
    const datoIzquierdo = datosOrdenados[indiceMitad - 1];
    const datoDerecho = datosOrdenados[indiceMitad];
    return (datoIzquierdo + datoDerecho) / 2;
  }
}

  3. La Moda

    Explicación

    La moda no tiene nada que ver con ropa en este contexto; es simplemente el valor o los valores que más se repiten en un conjunto de datos (es decir, el dato con mayor frecuencia absoluta). Sirve muchísimo en el análisis de datos para identificar tendencias, preferencias o los gustos más comunes de un grupo de usuarios.

    Fórmula Matemática

    A diferencia de otras medidas estadísticas, la moda no requiere de operaciones algebraicas complejas o una ecuación rígida. Se define conceptualmente a través de la frecuencia de los elementos:

    $$\text{Mo} = \{x_i \in X \mid f(x_i) = \max(f)\}$$

    Donde \(\text{Mo}\) es la moda, \(x_i\) representa los valores del conjunto, y \(f(x_i)\) es la frecuencia absoluta de dicho valor. Un conjunto puede ser:

    • Unimodal: Tiene una sola moda.
    • Bimodal o Multimodal: Tiene dos o más valores empatados con la máxima frecuencia.
    • Amodal: No tiene moda porque ningún valor se repite más que los otros.

    Cómo Hacerlo Tú Mismo

    Sigue estos pasos sencillos para encontrarla manualmente en tu cuaderno:

    • Paso 1: Mira detalladamente tu lista de datos. *(Opcional pero recomendado: ordénalos de menor a mayor para verlos mejor)*.
    • Paso 2: Cuenta cuántas veces aparece cada número dentro del conjunto.
    • Paso 3: Identifica cuál o cuáles números alcanzaron el récord de repeticiones. ¡Esa será tu moda!

    Ejemplos

    • Ejemplo 1 (Unimodal): Imagina que tienes la lista de datos \(\{5, 7, 5, 9, 5, 8\}\). Contamos las repeticiones: el \(7\), el \(9\) y el \(8\) aparecen una sola vez, mientras que el número \(5\) aparece tres veces. Como el \(5\) se repite más que cualquier otro, la moda es \(\text{Mo} = 5\).
    • Ejemplo 2 (Bimodal): En la lista \(\{3, 3, 3, 4, 6, 6, 6, 7\}\), tanto el \(3\) como el \(6\) aparecen tres veces. Al compartir el récord de repeticiones, este conjunto tiene dos modas: \(\text{Mo} = \{3, 6\}\).

    Cómo Programarlo

    Así es como una computadora procesa este algoritmo de conteo utilizando JavaScript :

    function calcularModaImperativa(lista) {
  let conteo = {};
  let maxVotos = 0;
  let modas = [];

  // 1. Contar las repeticiones de cada número
  for (let i = 0; i < lista.length; i++) {
    let num = lista[i];
    conteo[num] = (conteo[num] || 0) + 1;

    // Mantener actualizado cuál es el récord de repeticiones actual
    if (conteo[num] > maxVotos) {
      maxVotos = conteo[num];
    }
  }

  // 2. Juntar a todos los que alcanzaron el récord de repeticiones
  for (let num in conteo) {
    if (conteo[num] === maxVotos) {
      modas.push(Number(num));
    }
  }

  // Si todos tienen 1 repetición, o todos empataron por igual, es Amodal (no hay moda)
  if (maxVotos === 1 || modas.length === Object.keys(conteo).length) {
    return "No hay moda";
  }

  return modas.length === 1 ? modas[0] : modas;
}

  4. Porcentaje

    Explicación

    El porcentaje nos ayuda a entender qué tan grande es una parte en comparación con el todo, asumiendo que el "todo" siempre vale 100. Es una herramienta matemática vital en el desarrollo web para diseñar elementos como barras de progreso, calcular descuentos en comercios electrónicos o mostrar métricas de carga.

    Fórmula Matemática

    $$ \text{Porcentaje} = \left( \frac{\text{Parte}}{\text{Total}} \right) \times 100 $$

    ¿Cómo lo haces tú mismo?

    Sigue estos pasos para calcular cualquier porcentaje de forma manual en tu cuaderno:

    • Paso 1: Identifica y toma la "Parte" que deseas analizar de tu conjunto.
    • Paso 2: Divídela entre el "Total" posible para obtener un valor decimal.
    • Paso 3: Multiplica ese resultado decimal por 100 para convertirlo formalmente al formato de porcentaje.

    Ejemplos

    • Ejemplo 1 (Calificaciones de un examen): Imagina que obtuviste 45 puntos en una prueba cuyo valor máximo posible era de 50 puntos. Dividimos la parte entre el total: \(\frac{45}{50} = 0.90\). Luego, multiplicamos por 100: \(0.90 \times 100 = 90\%\). Tu rendimiento fue del \(90\%\).
    • Ejemplo 2 (Descuento en una tienda): Si un artículo cuesta originalmente \$80 y te descuentan \$20, puedes averiguar qué porcentaje representa ese descuento: \(\frac{20}{80} = 0.25\). Multiplicas por 100: \(0.25 \times 100 = 25\%\). Te hicieron un \(25\%\) de descuento.

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function calcularPorcentaje(puntosObtenidos, puntosTotales) {
  // 1. Dividimos los puntos logrados entre el total para obtener la fracción decimal
  // 2. Multiplicamos por 100 para transformarlo en un porcentaje (0% - 100%)
  return (puntosObtenidos / puntosTotales) * 100;
}

  5. Análisis de Tendencias

    Explicación

    Una tendencia es un patrón que nos dice hacia dónde se están moviendo las cosas a lo largo del tiempo. Es la base fundamental para realizar predicciones financieras, proyecciones de crecimiento de usuarios o estimaciones de rendimiento en cualquier sistema de software.

    Fórmula Matemática

    Para identificar la dirección del cambio entre dos periodos de tiempo consecutivos, evaluamos la tasa de variación: $$\Delta x = x_t - x_{t-1}$$

    Donde \(\Delta x\) es el cambio absoluto, \(x_t\) es el valor del periodo actual y \(x_{t-1}\) es el valor del periodo inmediato anterior. Si \(\Delta x > 0\) la tendencia sube, si \(\Delta x < 0\) la tendencia baja, y si \(\Delta x = 0\) se mantiene estable.

    ¿Cómo lo haces tú mismo?

    Sigue estos pasos para identificar la dirección de un patrón de datos de forma visual:

    • Ascendente (Al alza): Cada valor nuevo es mayor que el anterior (ej. ventas: \(100, 150, 200\)). ¡El negocio crece!
    • Descendente (A la baja): Cada valor nuevo es menor que el anterior (ej. usuarios: \(90, 75, 40\)). ¡Cuidado!
    • Estable: Los valores numéricos se mantienen muy parecidos o idénticos en el tiempo (ej. visitas: \(50, 51, 50\)).

    Ejemplos

    • Ejemplo de tendencia alcista: Un sitio web registra \([4, 6, 8]\) visitas por día. Al evaluar la variación final \(\Delta x = 8 - 6 = 2\), vemos que el resultado es positivo, lo que confirma un comportamiento al alza.
    • Ejemplo de tendencia bajista: Las temperaturas de una máquina bajan consecutivamente \([35, 30, 24]\). La variación final es \(\Delta x = 24 - 30 = -6\); al ser un número negativo, la tendencia va a la baja.

    Cómo Programarlo

    Así es como una computadora procesa este algoritmo de comparación utilizando JavaScript :

    function analizarTendencia(ventas) {
  // 1. Si no hay suficientes datos para comparar, no podemos definir una tendencia
  if (ventas.length < 2) return "Datos insuficientes";

  // 2. Ubicar dinámicamente el último y el penúltimo mes usando la longitud del arreglo
  const ultimoMes = ventas[ventas.length - 1];
  const mesAnterior = ventas[ventas.length - 2];

  // 3. Comparar los valores para determinar la tendencia
  if (ultimoMes > mesAnterior) {
    return "¡Tendencia al alza!";
  } else if (ultimoMes < mesAnterior) {
    return "Tendencia a la baja";
  } else {
    return "Tendencia estable";
  }
}

  6. Interpretación de Gráficos

    Explicación

    A nuestro cerebro le cuesta procesar miles de números crudos distribuidos en una tabla, pero entiende formas, proporciones y colores de manera instantánea. Un gráfico traduce datos matemáticos en dibujos y patrones visuales fáciles de asimilar.

    Fórmula Matemática

    Cada tipo de gráfico se rige por proporciones matemáticas exactas. Por ejemplo, en un gráfico de pastel, el ángulo en grados de cada rebanada se calcula multiplicando su frecuencia relativa por el total de la circunferencia: $$\theta = \frac{\text{Parte}}{\text{Total}} \times 360^\circ$$

    ¿Cómo lo haces tú mismo?

    Para interpretar y elegir el gráfico adecuado, debes identificar el objetivo de tu análisis de datos:

    • Gráfico de Barras: Úsalo para comparar cantidades individuales entre categorías diferentes (ej. ¿Qué materia tiene mejores notas?).
    • Gráfico de Líneas: Úsalo para observar la evolución temporal de una variable (ej. ¿Cómo cambió tu nota durante el año?).
    • Gráfico de Pastel: Úsalo para representar porcentajes y proporciones de un total absoluto (ej. ¿Qué fracción del día pasas estudiando vs jugando?).

    Ejemplos

    • Ejemplo de Barras: Si comparas las notas de Matemáticas (\(8\)) y Química (\(10\)), el gráfico mostrará una barra notablemente más alta para Química, permitiendo ver la diferencia de inmediato.
    • Ejemplo de Líneas: Al graficar tus notas mensuales \([6, 7, 9]\), la línea dibujará una diagonal ascendente, mostrando tu progreso visualmente.

    Cómo Programarlo

    En el desarrollo web, solemos estructurar la configuración de los gráficos mediante objetos de JavaScript utilizando librerías como Chart.js:

    const configGrafico = {
  type: 'line', // Define el dibujo: 'bar', 'line' o 'pie'
  data: {
    labels: ['Enero', 'Febrero', 'Marzo'],
    datasets: [{
      label: 'Rendimiento',
      data: [6, 7, 9]
    }]
  }
};

  7. Análisis de Datos

    Explicación

    El análisis de datos no es magia, es un proceso metodológico estructurado paso a paso para extraer información real, verídica y útil a partir de números sueltos. Como desarrolladores, creamos algoritmos y sistemas automatizados que ejecutan este ciclo continuamente.

    Fórmula Matemática

    El proceso cuantifica la calidad de la información mediante métricas como la t ocultotros filtrados: $$\text{Eficiencia} = \frac{\text{Datos Limpios}}{\text{Datos Recolectados}} \times 100$$

    ¿Cómo lo haces tú mismo?

    Puedes aplicar de forma manual este flujo analítico en cualquier proyecto o problema cotidiano:

    • 1. Recolección: Conseguir los datos brutos desde las fuentes disponibles (ej. pedir las notas en papel a tus profesores).
    • 2. Limpieza: Quitar registros duplicados, nulos o erróneos (ej. borrar un renglón que dice nota "999" por una falla de tipeo).
    • 3. Procesamiento: Realizar todas las operaciones matemáticas necesarias (ej. calcular promedios, medianas o porcentajes).
    • 4. Visualización: Traducir esos resultados numéricos finales en componentes visuales (ej. dibujar los gráficos de líneas o barras).
    • 5. Interpretación: Tomar decisiones lógicas basadas en la evidencia obtenida (ej. "Mi promedio bajó en Física, debo estudiar más ahí").

    Ejemplos

    • Caso práctico: Recolectas las notas \([8, 10, 999, 9]\). En la limpieza eliminas el \(999\). En el procesamiento calculas el promedio de los datos válidos: \(\frac{8+10+9}{3} = 9\). Lo visualizas y concluyes que tu rendimiento es excelente.

    Cómo Programarlo

    Este es un ejemplo de cómo programar en JavaScript las etapas de limpieza (filtrado) y procesamiento (cálculo de media) sobre un arreglo:

    function procesarDatosEstatidicos(datosIncompletos) {
  // 1. Etapa de Limpieza: Filtramos y quitamos notas imposibles (ej. mayores a 10)
  const datosLimpios = datosIncompletos.filter(nota => nota >= 0 && nota <= 10);

  // 2. Etapa de Procesamiento: Calculamos la media con los datos válidos
  const suma = datosLimpios.reduce((acc, curr) => acc + curr, 0);
  return datosLimpios.length > 0 ? suma / datosLimpios.length : 0;
}

  8. Detección de Anomalías

    Explicación

    Una anomalía es un dato atípico que se aleja de forma extrema del comportamiento normal del resto del grupo. Detectarlos es crucial en la informática para identificar fraudes con tarjetas de crédito, intrusos en redes de servidores o errores graves de medición.

    Fórmula Matemática

    Un método estadístico estándar consiste en marcar como anomalía cualquier dato (\(x_i\)) que se encuentre fuera del intervalo delimitado por varias desviaciones estándar (\(\sigma\)) respecto a la media (\(\bar{x}\)): $$\text{Anomalía} \iff x_i < \bar{x} - 2\sigma \quad \text{ó} \quad x_i > \bar{x} + 2\sigma$$

    Donde la desviación estándar se calcula como: $$\sigma = \sqrt{\frac{\sum_{i=1}^n (x_i - \bar{x})^2}{n}}$$

    ¿Cómo lo haces tú mismo?

    Puedes detectar datos anómalos de manera lógica en tu cuaderno siguiendo estos pasos:

    • Paso 1: Observa tus datos normales y define su rango regular (ej. \([20, 21, 22, 20]\) se mueven alrededor de \(21\)).
    • Paso 2: Examina si aparece un valor radicalmente alejado del grupo (ej. un valor de \(45\)).
    • Paso 3: Compara ese dato sospechoso contra la dispersión típica. Al romper drásticamente el patrón, lo clasificas como anomalía.

    Ejemplos

    • Ejemplo en temperaturas: Si un sensor registra \([22, 23, 22, 85, 23]\) grados, el número \(85\) representa una anomalía evidente, probablemente ocasionada por una falla eléctrica o un cortocircuito.

    Cómo Programarlo

    Así es como un sistema calcula de forma exacta la media y la desviación estándar en JavaScript para aislar los valores atípicos:

    function detectarAnomaliasTemperatura(temperaturas) {
  if (temperaturas.length < 3) return "Datos insuficientes para analizar";

  // 1. Calcular el Promedio (Media)
  let suma = 0;
  for (let i = 0; i < temperaturas.length; i++) {
    suma += temperaturas[i];
  }
  const promedio = suma / temperaturas.length;

  // 2. Calcular la Desviación Estándar (Qué tanto varían los datos normalmente)
  let sumaDiferenciasCuadrado = 0;
  for (let i = 0; i < temperaturas.length; i++) {
    sumaDiferenciasCuadrado += Math.pow(temperaturas[i] - promedio, 2);
  }
  const varianza = sumaDiferenciasCuadrado / temperaturas.length;
  const desviacionEstandar = Math.sqrt(varianza);

  // 3. Definir el límite inteligente (Promedio + 2 veces la desviación)
  const limiteSuperior = promedio + (2 * desviacionEstandar);
  const limiteInferior = promedio - (2 * desviacionEstandar);

  // 4. Filtrar las anomalías
  let anomalias = [];
  for (let i = 0; i < temperaturas.length; i++) {
    if (temperaturas[i] > limiteSuperior || temperaturas[i] < limiteInferior) {
      anomalias.push(temperaturas[i]);
    }
  }

  return anomalias;
}
