  1. Promedio y Media Aritmética

    Explicación

    La media y el promedio son exactamente lo mismo.
    En el día a día usamos la palabra "promedio", pero en el mundo de las matemáticas y la estadística, los profesionales prefieren llamarle "media" (o formalmente, media aritmética). Es el valor que se obtiene al sumar todos los datos de un conjunto y luego dividir el resultado entre el número total de datos.

    Fórmula

    $$\bar{x} = \frac{\sum_{i=1}^n x_i}{n}$$

    ¿Qué significa cada símbolo?

    • \(\bar{x}\) (se lee "x barra"): Representa la media aritmética o promedio.
    • \(\sum\) (letra griega Sigma): Es el símbolo de sumatoria, lo que significa que debes sumar todos los valores desde el primero (\(i=1\)) hasta el último (\(n\)).
    • \(x_i\): Representa cada uno de los valores individuales en tu conjunto de datos.
    • \(n\): Es el número total de datos que estás sumando.

    ¿Cómo lo haces tú mismo?

    Sigue estos pasos para que no te pierdas al momento de quererlo hacer tú mismo en un cuaderno:

    Imaginemos que los datos que tienes son: \([8, 9, 7]\)

    1. Suma todos los datos -> \(8 + 9 + 7 = 24\)
    2. Cuenta la cantidad de tus datos -> \([8, 9, 7] = 3\) datos
    3. Ahora divides la suma total entre la cantidad de datos -> \(\frac{24}{3} = 8\)
    4. ¡Listo! Ya tienes tu media aritmética (promedio), que es por supuesto \(8\)

    Ejemplos

    Ejemplo 1: El promedio de tus calificaciones

    Imagina que quieres saber cuál fue tu rendimiento final en una materia y tus notas de los cuatro periodos del año fueron: 8, 9, 7 y 10.

    1. Sumas todas las notas: \(8 + 9 + 7 + 10 = 34\)
    2. Divides entre el número total de notas: \(\frac{34}{4} = 8.5\)
    3. Resultado: La media aritmética de tus calificaciones es \(8.5\)
    Ejemplo 2: Los puntos anotados por un jugador de baloncesto

    Un jugador de básquetbol quiere saber cuántos puntos mete por partido en promedio. En sus últimos 5 juegos anotó: 12, 18, 15, 20 y 10 puntos.

    1. Sumas los puntos de todos los partidos: \(12 + 18 + 15 + 20 + 10 = 75\)
    2. Divides entre el número total de partidos: \(\frac{75}{5} = 15\)
    3. Resultado: La media aritmética es \(15\) puntos por partido.
    Ejemplo 3: El dinero que gastas en la semana

    Quieres calcular cuánto gastas típicamente en transporte de lunes a viernes. Tus gastos diarios fueron: $2.50, $3.00, $2.50, $4.00 y $3.00.

    1. Sumas el dinero de los 5 días: \(2.50 + 3.00 + 2.50 + 4.00 + 3.00 = 15.00\)
    2. Divides entre los 5 días de la semana: \(\frac{15.00}{5} = 3.00\)
    3. Resultado: La media aritmética de tu gasto diario es de \(\$3.00\)

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function calcularMedia(arr) {
  // 1. Sumamos todo
  let suma = 0;
  for (let i = 0; i < arr.length; i++) {
    suma += arr[i];
  }
  // 2. Dividimos por la cantidad de elementos
  const media = suma / arr.length;
  return media;
}

  2. Mediana

    La mediana es el valor central de un conjunto de datos ordenados de menor a mayor. Divide la distribución exactamente por la mitad, dejando un 50% de los valores por debajo y un 50% por encima. Es una medida de tendencia central clave en estadística

    ¿Cómo lo haces tú mismo?

    • Paso 1: Ordena tus datos de menor a mayor.
    • Paso 2: Cuenta su distancia.
    • Paso 3: Si es impar la distancia: La media es exactamente el numero que este en la mitad.
    • Paso 4: Si es par la distancia: La mediana es el promedio de los dos números centrales.

    Ejemplos

    • Cantidad impar = {3, 5, 8, 10, 11}. El número central es 8.
    • Cantidad par = {2, 4, 6, 8}. Los dos centrales son 4 y 6, por lo que el promedio es \(\frac{4+6}{2}\) = 5

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function calcularMediana(listaNumeros) {
  // 1. Ordenar los datos de menor a mayor sin alterar la lista original
  const datosOrdenados = [...listaNumeros].sort((a, b) => a - b);
  const n = datosOrdenados.length;

  // 2. Encontrar el índice de la mitad exacta
  const indiceMitad = Math.floor(n / 2);

  // 3. Si la cantidad de datos es IMPAR, el del centro es la mediana
  if (n % 2 !== 0) {
    return datosOrdenados[indiceMitad];
  }
  // 4. Si es PAR, se promedian los dos elementos del centro
  else {
    const datoIzquierdo = datosOrdenados[indiceMitad - 1];
    const datoDerecho = datosOrdenados[indiceMitad];
    return (datoIzquierdo + datoDerecho) / 2;
  }
}

  3. Moda

    La moda no tiene nada que ver con ropa en este contexto; es simplemente el valor que más se repite en un conjunto de datos. Sirve muchísimo para saber cuáles son los gustos más comunes de los usuarios.

    ¿Cómo lo haces tú mismo?

    • Mira tu lista de datos: 5, 7, 5, 9, 5, 8.
    • Cuenta cuántas veces aparece cada número.
    • El 5 aparece tres veces, más que cualquier otro. ¡Esa es tu moda!

      • BUSCADOR DE MODA AUTOMÁTICO:

      Ingresa una serie de números para ver cuál se repite más:

      MODA:

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function calcularModaImperativa(lista) {
  let conteo = {};
  let maxVotos = 0;
  let modas = [];

  // 1. Contar los votos de cada número
  for (let i = 0; i < lista.length; i++) {
    let num = lista[i];
    conteo[num] = (conteo[num] || 0) + 1;

    // Mantener actualizado cuál es el récord de votos actual
    if (conteo[num] > maxVotos) {
      maxVotos = conteo[num];
    }
  }

  // 2. Juntar a todos los que alcanzaron el récord de votos
  for (let num in conteo) {
    if (conteo[num] === maxVotos) {
      modas.push(Number(num));
    }
  }

  // Si todos tienen 1 voto, o todos empataron igual, no hay moda (Amodal)
  if (maxVotos === 1 || modas.length === Object.keys(conteo).length) {
    return "No hay moda";
  }

  return modas.length === 1 ? modas[0] : modas;
}

  4. Porcentaje

    El porcentaje nos ayuda a entender qué tan grande es una parte en comparación con el todo, asumiendo que el "todo" siempre vale 100. Es vital para barras de progreso o descuentos.

    ¿Cómo lo calculas a mano?

    $$ \text{Porcentaje} = \left( \frac{\text{Parte}}{\text{Total}} \right) \times 100 $$

    • Paso 1: Toma la "Parte" que quieres analizar. Ejemplo: Tienes 45 puntos en un examen.
    • Paso 2: Divídelo por el "Total" posible. El examen valía 50 puntos (45 / 50 = 0.90).
    • Paso 3: Multiplica por 100 para pasarlo a formato de porcentaje (0.90 * 100 = 90%).

      • SIMULADOR DE EFECTIVIDAD / AVANCE:

      Calcula el porcentaje de forma instantánea:

      PORCENTAJE:

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function calcularPorcentaje(puntosObtenidos, puntosTotales) {
  // 1. Dividimos los puntos logrados entre el total para obtener la fracción decimal
  // 2. Multiplicamos por 100 para transformarlo en un porcentaje entero (0% - 100%)
  return (puntosObtenidos / puntosTotales) * 100;
}

  5. Análisis de Tendencias

    Una tendencia es un patrón. Nos dice hacia dónde se están moviendo las cosas a lo largo del tiempo. Es la base de las predicciones financieras y de crecimiento.

    ¿Cómo identificarla a simple vista?

    • Ascendente: Cada valor nuevo es mayor que el anterior (ej. ventas: 100, 150, 200). ¡El negocio crece!
    • Descendente: Cada valor nuevo es menor. ¡Cuidado!
    • Estable: Los valores se mantienen muy parecidos.

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function analizarTendencia(ventas) {
  // 1. Si no hay suficientes datos para comparar, no podemos definir una tendencia
  if (ventas.length < 2) return "Datos insuficientes";

  // 2. Ubicar dinámicamente el último y el penúltimo mes usando la longitud del arreglo
  const ultimoMes = ventas[ventas.length - 1];
  const mesAnterior = ventas[ventas.length - 2];

  // 3. Comparar los valores para determinar la tendencia
  if (ultimoMes > mesAnterior) {
    return "¡Tendencia al alza!";
  } else if (ultimoMes < mesAnterior) {
    return "Tendencia a la baja";
  } else {
    return "Tendencia estable";
  }
}
    Análisis de Tendencias: Permite identificar la dirección (alcista o bajista) de una serie de datos a lo largo del tiempo.

  6. Interpretación de Gráficos

    A nuestro cerebro le cuesta leer miles de números en una tabla, pero entiende formas y colores al instante. Un gráfico traduce datos en dibujos.

    ¿Cómo elegir y leer el correcto?

    • Gráfico de Barras: Úsalo para comparar cantidades (¿Qué materia tiene mejores notas?).
    • Gráfico de Líneas: Úsalo para ver el tiempo (¿Cómo evolucionó tu nota durante el año?).
    • Gráfico de Pastel: Úsalo para ver porcentajes (¿Qué parte del día pasas estudiando vs jugando?).

  7. Análisis de Datos

    No es magia, es un proceso paso a paso para sacarle la verdad a los números. Como desarrolladores, creamos sistemas que hacen esto automáticamente.

  8. Detección de Anomalías

    Una anomalía es el "bicho raro" del grupo. Es un dato que se aleja tanto de lo normal que nos obliga a investigar si hubo un error o un evento especial.

    ¿Cómo detectarlo manualmente?

    • Mira tus datos normales: 20, 21, 22, 20. Sabes que el rango es alrededor de 21.
    • Aparece un 45.
    • Comparas el 45 con tu rango normal. Al estar tan lejos, lo marcas como anomalía.

      Ingresa notas (rango esperado 0-10). Si pones un error (ej: 88), el sistema lo aislará:

    ¿Cómo lo hace la computadora? Usando JavaScript 

    function detectarAnomaliasTemperatura(temperaturas) {
  if (temperaturas.length < 3) return "Datos insuficientes para analizar";

  // 1. Calcular el Promedio (Media)
  let suma = 0;
  for (let i = 0; i < temperaturas.length; i++) {
    suma += temperaturas[i];
  }
  const promedio = suma / temperaturas.length;

  // 2. Calcular la Desviación Estándar (Qué tanto varían los datos normalmente)
  let sumaDiferenciasCuadrado = 0;
  for (let i = 0; i < temperaturas.length; i++) {
    sumaDiferenciasCuadrado += Math.pow(temperaturas[i] - promedio, 2);
  }
  const varianza = sumaDiferenciasCuadrado / temperaturas.length;
  const desviacionEstandar = Math.sqrt(varianza);

  // 3. Definir el límite inteligente (Promedio + 2 veces la desviación)
  // El "2" es el factor de tolerancia. Más de 2 ya es muy raro.
  const limiteSuperior = promedio + (2 * desviacionEstandar);
  const limiteInferior = promedio - (2 * desviacionEstandar);

  // 4. Filtrar las anomalías
  let anomalias = [];
  for (let i = 0; i < temperaturas.length; i++) {
    if (temperaturas[i] > limiteSuperior || temperaturas[i] < limiteInferior) {
      anomalias.push(temperaturas[i]);
    }
  }

  return anomalias;
}
